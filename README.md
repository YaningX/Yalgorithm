# Yalgorithm
============
## What's about
- Common algorithm
- data structure
- leetcode problems

## What's included
```
Yalgorithm/
├── data-structure/
│   └── btree/
│       ├── BTNode
│       ├── BTree
│       ├── AbstractBTree
│       ├── IterBTree
│       └── RecBTree
├── geeks4geeks/
│   └── findMaxProduct
├── sort/
│       ├── CountingSort (非比较排序)
│       ├── BucketSort (非比较排序)
│       ├── RadixSort (非比较排序)
│       ├── MergeSort (比较排序)
│       ├── Qsort (比较排序)
│       ├── HeapSort (比较排序)
│       ├── BubbleSort (比较排序)
│       └── SelectSort (比较排序)
└── backtracking/
    ├── DiceRollGame
    ├── Key2String
    ├── Permutation
    └── DPSubSetSum
```
## Maven Plugin
- Code quality is checked by maven plugins: checkstyle, findbugs.
- JavaDoc is generated by maven plugin:javadoc.
